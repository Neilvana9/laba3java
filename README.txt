Библиотека

Современная система управления библиотекой, с функионалом, который позволяет отслеживать книги, авторов, читателей, штрафы за просрочку возврата книг, просматривать и оставлять отзывы, рейтинг книг и авторов. Система построена на Spring Boot с использованием PostgreSQL для хранения данных и развертывается в Docker контейнерах.


Основные сущности:
 Author (Автор) - информация об авторах книг
 Book (Книга) - информация о книгах, их авторах и статусе
 Reader (Читатель) - информация о читателях и их активных заимствованиях
 Fine (Штраф) - информация о штрафах за просрочку возврата книг

Связи между сущностями:
 Книга-Автор: связь многие-ко-многим (одна книга может иметь нескольких авторов)
 Книга-Читатель: связь многие-к-одному (одна книга может быть у одного читателя)
 Штраф-Читатель: связь многие-к-одному (один читатель может иметь несколько штрафов)
 Штраф-Книга: связь многие-к-одному (одна книга может быть связана с несколькими штрафами)

Операции сервиса:
	CRUD операции:
	 Авторы: создание, чтение, обновление, удаление
	 Книги: создание, чтение, обновление, удаление
	 Читатели: создание, чтение, обновление, удаление
	 Штрафы: создание, чтение, обновление, удаление

	Операции:
	 1. Выдача книги читателю - читатель может взять книгу на 30 дней
	 2. Возврат книги - при возврате с просрочкой начисляется штраф
	 3. Поиск книг по автору - получение всех книг конкретного автора
	 4. Получение штрафов читателя - просмотр всех штрафов конкретного читателя
	 5. Получение взятых книг - просмотр всех книг, которые в данный момент находятся у читателя
	 6. Оплата всех штрафов - оплата всех неоплаченных штрафов конкретного читателя

	Бизнес-операции:
	 1. Просмотр статистики - получение списка самых популярных книг по количеству взятий
	 2. Оставить отзыв о книге - читатель может оценить книгу от 1 до 5 и оставить комментарий
	 3. Оставить отзыв об авторе - читатель может оценить работы автора
	 4. Просмотреть отзывы - просмотреть все отзывы о книге или авторе
	 5. Верификация отзыва - подтверждение отзыва библиотекарем
	 6. Аналитика - просмотр статистики по библиотеке (средняя продолжительность чтения, процент возврата вовремя)
	 7. Рекомендации - получение персонализированных рекомендаций книг на основе рейтингов



Запуск контейнера:
docker run -d --name library-postgres -e POSTGRES_DB=library -e POSTGRES_USER=user -e POSTGRES_PASSWORD=password -p 5432:5432 postgres:15

Проверка:
docker ps

Остановка контейнера: 
docker-compose down
